## 4) ⚙️ Як працює (потік)
- **Ручний режим:** числа з URL-параметрів. Зміна = нове посилання.
- **Живий режим (multi-tenant):** стрімер реєструється на `connect.html` (Ko-fi-токен + капча) → отримує `id`. Далі: `💸 Ko-fi-донат → 🔔 webhook?id= → ☁️ Worker (+сума в goal:<id>) → 📊 віджет опитує /total?id= → смужка оновлюється сама`. Токен можна змінити без зміни id (`/update-token`), скид перед стрімом — `/reset?id=&key=`.

## 5) 🚀 Як користуватись (стрімер)
1. **Ручний:** конструктор → налаштуй вигляд → «Скопіювати» → OBS *Browser Source* (≈600×150).
2. **Живий:** `connect.html` → пройди капчу → встав свій Ko-fi Verification Token → отримай 3 посилання (віджет/webhook/скид) → webhook у Ko-fi, посилання в OBS. Кнопка «Налаштувати вигляд» веде в конструктор зі своїм `id`.
3. Навігація-перемикач (Конструктор ⇄ Підключити Ko-fi) — угорі кожної сторінки.
