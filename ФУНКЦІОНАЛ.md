## 1) 🎛️ Віджет `goal.html` — параметри URL
| Параметр | Значення | За замовч. |
|---|---|---|
| `label` | Назва над смужкою | `Donation Goal` |
| `current` / `goal` | Поточне / ціль (ручний режим) | `0` / `100` |
| `prefix` / `suffix` | Текст до/після чисел (`$`, `subs`) | — |
| `theme` | `teal/coral/sun/lilac/green` | `teal` |
| `color` / `bg` / `text` | свій колір смужки / доріжки / тексту | — / напівпрозор. / `#FFF` |
| `height` | Висота, px | `30` |
| `style` | `rounded` / `flat` / `striped` | `rounded` |
| `milestones` | Позначки-етапи (через кому) | — |
| `done` | Повідомлення при 100% | — |
| `showvalues` | `1`/`0` | `1` |
| ⭐ `api` | **Живий режим:** URL до `/total` → опитує кожні 7с | — (без нього = ручний) |

## 4) ⚙️ Як працює (потік)
- **Ручний режим (v1):** числа з URL-параметрів. Зміна = нове посилання.
- **Живий режим (v2):** `💸 Ko-fi-донат → 🔔 webhook → ☁️ Worker (+сума в KV) → 📊 віджет опитує /total → смужка оновлюється сама`.

## 5) 🚀 Як користуватись (стрімер)
1. Відкрий конструктор → налаштуй → (опц.) увімкни «живий режим».
2. **Скопіювати посилання** → OBS → Browser Source (≈600×150).
3. Живий режим: один раз підключити Ko-fi webhook (URL `…/webhook` + токен через `wrangler secret put` або dashboard).
